# Portfolio Website

Modern portfolio website built with Vue 3 (Frontend) and Express/SQLite (Backend).

## Repository Structure

- `client/`: Vue 3 Frontend Application
- `server/`: Express Backend API and SQLite Database
- `uploads/`: Project images

## Setup

1. **Frontend**:
    ```bash
    cd client
    npm install
    npm run dev
    ```

2. **Backend**:
    ```bash
    cd server
    npm install
    node index.js
    ```

## Features
- Dynamic Portfolio Display
- Admin Dashboard for Uploads
- Modern Glassmorphism UI
